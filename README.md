## 🎯 Project Overview
수많은 영양제 정보 속에서 사용자가 겪는 결정 장애를 해결하기 위해 기획된 **AI 에이전트 협업 시스템**입니다. 본 MVP는 인지적 노동을 간소화하고, 명확한 제약 조건(약물 상호작용, 알레르기)을 수행하여 사용자에게 최적의 제품 1종을 제안하는 것에 집중합니다.

## 🚀 Key Points
* **Multi-agents system**: 총 5개의 에이전트를 사용하여 입력 -> 분석(step 1) -> 검색(step 2) -> 최적화 및 검증(step 3) -> 재고 확인(step 4) -> 최종 응답에 이르는 플로우 설계
* **안전한 판단 근거 수집**: 건강 목표는 물론, 나이, 성별, 기존 질환 및 알러지 등을 조사하여 안전한 제품 추천
* **최적의 결론 도출**: 건강 목표 뿐 아니라, 사용자 예산 범위를 고려하여 최적의 제품을 최종 추천

## 🤖 Agent Workflow & Architecture
사용자의 인지적 부하를 줄이기 위해 5개의 특화된 에이전트가 협업합니다. 
1. **Intake & Analysis Agent (분석)**
   - 사용자의 건강 목표, 질병 이력, 예산, 성별, 나이, 알러지 여부 등을 분석하여 필요한 핵심 성분과 에이전트의 조언을 도출합니다. 더하여, 검색 에이전트가 필요한 쿼리를 추출합니다. 
2. **Search Agent (검색)**
   - 선정된 성분을 바탕으로 네이버 쇼핑에서 실제 구매 가능한 제품 리스트를 실시간으로 수집합니다.
3. **Optimizer Agent (최종 가성비 One 선택)**
   - 수집된 제품 중 가성비와 성분 함량을 비교하여 최적의 제품 1종, 혹은 최적의 제품 조합을 최종 선정합니다. 
4. **Verifier Agent (Compliance)**
   - 선정된 제품이 사용자의 복용 약물 및 알레르기와 충돌하는지 검증하고 주의사항을 제공합니다.
5. **Connecting Agent (유저 연결)**
   - Optimizer에 의해 선정된 제품의 페이지에서 구매 가능 여부를 크롤링하고, 이를 통해 품절 여부를 짐작하여 제안합니다.
